### Projeto

O <b>Next Project</b> foi criado com o propósito de organizar projetos e atividades de maneira fácil e simples 

### Layout

![next-project](https://user-images.githubusercontent.com/60704988/78832820-49faa200-79c2-11ea-9a67-ec483b6063af.gif)

### Funcionamento

<b>Tela Inicial</b>
- Visualização de projetos cadastrados
- Login em projeto via ID
- Cadastro de novo projeto

<b>Tela de Atividades</b>
- Visualização de informações do projeto
- Edição e exclusão do projeto
- Visualização, cadastro, edição e exclusão de atividades
- Logout

### Acesso

A aplicação está disponível [neste link](https://nextprojectchallenge.herokuapp.com/)

Também pode ser acessada rodando `npm start` tanto no diretório `backend` quanto `frontend` podendo assim ser visualizada em `http://localhost:3000/`
